* **Use After Free Vulnerability (High Severity):** CVE-2023-5186 highlights a use after free vulnerability in Passwords, potentially leading to heap corruption and code execution if an attacker convinces the user to interact with specially crafted content.

It is strongly recommended that users update Google Chrome to version **117.0.5938.132 or later** to mitigate these vulnerabilities. This patch addresses all the mentioned CVEs and significantly enhances browser security.
